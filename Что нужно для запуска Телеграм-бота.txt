1. Установите приложение Телеграм на Ваш PC, либо на любой мобильный девайс который поддерживает это приложение или зайдите в него через web версию.
2. Обратитесь к @BotFather и с помощью специальных команд создайте бота, команды предоставляет @BotFather.
3. Получите api-ключ Яндекс переводчика. Ссылка для получения api-ключа: 	https://tech.yandex.ru/translate/.
4. Установите на свой PC NodeJS, если он у Вас уже установлен, то повторная установка не требуется.
5. Скачайте данный проект с github.
6. Откройте файл index.js в любом редакторе кода и внесите необходимые изменения:
   в 6 строке: const token = 'Токен (api) Телеграм-бота'; 
между одинарными кавычками вставте api который Вы получили при создании Телеграм-бота в диалоге с @BotFather.
   В 13 строке: const key = 'api-ключ Яндекс переводчика'; 
между одинарными кавычками вставте api-ключ который Вы получили от Яндекс.
7. Зайдите в папку с проектом через командную строку, либо зажмите клавишу Shift и кликните правой кнопкой мыши по папке в которой находится проект, выберете пункт меню "Открыть окно команд".
8. В командной строке пропишите npm install и нажмите клавишу Enter. 
9. По окончании всех действий по установке дополнительных пакетов (см.п.8) используемых в проекте, в этой же командной строке пропишите npm start и нажмите клавишу Enter.
10.В приложении Телеграм найдите созданного Вами Телеграм-бота и напишите ему сообщение на английском языке, в ответ от него Вы получите сообщение с переводом текста, который Вы ранее ему отправили.
